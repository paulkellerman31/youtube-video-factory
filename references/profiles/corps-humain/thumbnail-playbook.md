# Thumbnail Playbook — chaîne Corps Humain

Hérite du playbook universel (`references/thumbnail-playbook.md`). Ce fichier ne fait que
remplir les **NICHE SLOTS** pour la chaîne corps-humain ; tout le reste (postage-stamp test,
formule CTR, archétypes, spec overlay) reste défini dans l'universel.

## NICHE SLOTS — corps-humain
```
- Brand palette + HEX: cyan #19D3DA + rouge sang #E03131 sur noir profond. Pas de fond blanc
  d'hopital plat, pas de vert clinique fade.
- Hero objects (3-5 iconiques): organe surdimensionne (coeur, cerveau, poumon), cellule/
  neurone en gros plan, structure microscopique (bacterie, ADN), coupe anatomique 3D.
- Aesthetic / setting: rendu medical 3D photorealiste, rim light clinique, particules fines,
  faible profondeur de champ, objet hero centre-gauche, espace negatif a droite pour le texte.
- Banned: gore sanglant a outrance, visages reconnaissables, texte/labels dans l'image IA.
- Default archetype mix: Contrast Grab + Drama/Exaggeration en lead (fait choc anatomique) ;
  Transformation pour les avant/apres (maladie, vieillissement, effort).
```

## Notes
- Texte overlay <= 3 mots, ALL CAPS, pose en post (Canva) — jamais dans le prompt IA.
- Ligne 1 blanc / ligne 2 cyan #19D3DA. Jamais dans les 20 % du bas.
- L'organe/cellule héros doit passer le postage-stamp test : surdimensionné, ultra net, un seul
  point focal.
