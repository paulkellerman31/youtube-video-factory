# Thumbnail Playbook — chaîne Corps Humain

Hérite du playbook universel (`references/thumbnail-playbook.md`). Ce fichier remplit juste les
**NICHE SLOTS** pour la chaîne corps-humain ; le reste (postage-stamp test, formule CTR,
archétypes, spec overlay) reste défini dans l'universel.

## NICHE SLOTS — corps-humain
```
- Brand palette + HEX: teal #2EC4B6 + corail #FF6B6B sur creme #FFF4E0. Lumineux, accueillant.
- Hero objects (3-5 iconiques): organe-personnage souriant (coeur, cerveau, rein), grosse
  goutte d'eau a tete sympathique, cellule mignonne, petit personnage humain stylise.
- Aesthetic / setting: cartoon 2D plat facon TED-Ed, contours nets, aplats, formes simples,
  expression marquee (surprise / inquietude / soulagement), objet hero centre-gauche,
  espace negatif a droite pour le texte.
- Banned: photorealisme, 3D, gore, rendu sombre, texte/labels dans l'image IA, visage realiste.
- Default archetype mix: Emotional Hook (expression cartoon forte) + Contrast Grab en lead ;
  Transformation pour les avant/apres.
```

## Notes
- Texte overlay <= 3 mots, ALL CAPS, posé en post (Canva) — jamais dans le prompt IA.
- Ligne 1 blanc ou crème / ligne 2 corail #FF6B6B (contraste sur fond clair). Jamais dans les
  20 % du bas.
- L'objet/personnage héros doit passer le postage-stamp test : grand, expressif, un seul point
  focal, contour net.
