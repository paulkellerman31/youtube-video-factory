universel par design. Ici, seuls les NICHE SLOTS (§8) changent.

## NICHE SLOTS — à remplir avant la première vidéo
```
- Brand palette + HEX:        (or chaud + ombre — HEX à verrouiller)
- Hero object library (3–5):  (ex: casque de centurion, aigle de légion, sesterce, glaive, buste de marbre)
- Aesthetic notes / setting:  (forum au crépuscule, colonnade torchée, carte parchemin)
- Banned elements:            (visages identifiables, éléments modernes, néon)
- Default archetype mix:      (Urgency/FOMO + Drama lead ; Transformation pour ascension/chute)
```
