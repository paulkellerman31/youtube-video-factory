# Projet d'exemple

Squelette montrant les 3 fichiers d'entrée que le skill écrit pour chaque vidéo :
`project-config.json`, `image-prompts.json`, `voiceover.txt`.

Les vrais projets vivent dans `projects/<channel>/<date>_<slug>/` et sont **gitignored**
(ils contiennent tes vidéos rendues). Lance le rendu avec : `factory run projects/_example`.
